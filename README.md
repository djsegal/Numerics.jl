### NumericsBundle.jl
##### Bundle of Julia packages relevant to numerics

---

To load use:

```julia
] add https://github.com/djsegal/NumericsBundle.jl
using NumericsBundle
```

You should now have the following packages available to you:

+ CSV
+ DataFrames
+ DataStructures
+ FFTW
+ LinearAlgebra
+ Statistics
+ StatsBase
+ Plots
